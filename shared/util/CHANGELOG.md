## [Unreleased]

### Added

- 初始化包骨架
- `colorToRGB()`：颜色解析（支持 #RGB/#RRGGBB/#RRGGBBAA/rgb()/rgba()）
- `generateUid()`：唯一 ID 生成（时间戳 + 计数器 + 随机数）
- `round()`：四舍五入到指定小数位数
- `distance()`：两点欧几里得距离
- `normalize()`：绝对坐标归一化到 0-1
- `denormalize()`：归一化坐标还原为绝对坐标
